👤 Profile Card

A clean and responsive Profile Card UI built using pure HTML and CSS.
This project displays a personal profile with an image, name, role, and social media links.


🚀 Features

Responsive centered layout

Circular profile image with styled border

Social media buttons with brand colors

Modern font using Google Fonts (Poppins)

Boxicons integration for social icons

Clean shadowed card design


🛠 Technologies Used

HTML5

CSS3 (Flexbox)

Google Fonts

Boxicons
